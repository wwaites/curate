CKAN Curation Tool
==================

This package contains :program:`curate` a tool for checking `CKAN`_
packages to see if they meet certain criteria, for example for
inclusion in certain groups.

Relevant places to go for information are:

  * Documentation (this page): http://packages.python.org/curate
  * Source Code: http://bitbucket.org/okfn/curate
  * Wiki Page: http://ckan.org/wiki/curate
